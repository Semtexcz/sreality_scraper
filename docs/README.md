# Docs

This directory contains technical documentation for `scraperweb`.

- `ARCHITECTURE.md`: runtime and package structure overview
- `MODULES.md`: package/module responsibilities
- `ARTIFACTS.md`: repo-level files and generated outputs
- `ARTIFACT_REFERENCE.md`: field-level reference for raw, normalized, and
  enriched JSON artifact contracts
- `ANALYSIS_NOTEBOOK_PLAN.md`: staged plan for the exploratory Jupyter
  notebook, price-surface analysis, uncertainty reporting, and the later
  predictive-model training stage
- `ANALYSIS_DATASET_CONTRACT.md`: approved first contract for the canonical
  notebook and training analysis dataset export
- `PRICE_SURFACE_WORKFLOW.md`: approved first design for the grid-based
  price-surface workflow, precision-aware inclusion rules, and uncertainty
  outputs
- `ROADMAP.md`: short-term improvement plan
- `TASK_SEQUENCE.md`: recommended implementation order for the current backlog
  task sequence
- `api/`: reserved location for generated HTML API documentation
- `uml/`: lightweight architecture diagrams or placeholders
